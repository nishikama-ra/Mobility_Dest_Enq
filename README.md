# 移動しやすいまち西鎌倉 投稿ボード

地域の「こんな移動があると助かる」を集めるための、GitHub Pages + Google Apps Script + Google スプレッドシート構成のWebアプリです。

## できること

- 住民がスマホから移動ニーズを投稿できます。
- 投稿時に「公開してよい」または「事務局だけに知らせる」を選べます。
- 公開希望の投稿は、事務局が内容を確認し、承認したものだけがWebページに表示されます。
- 事務局はGoogleスプレッドシートですべての投稿を確認できます。
- 投稿、訂正、取消時に投稿者本人と事務局へメール通知できます。
- 投稿者はメールで届く確認・変更用リンクから、自分の投稿を確認・訂正・取消できます。
- メールアドレス、記入者、確認・変更用リンク、事務局メモはWeb公開されません。

## 構成

- `index.html`, `styles.css`, `app.js`: GitHub Pagesで公開する住民向け画面
- `assets/mobility-map.png`: ヒーロー画像
- `gas/Config.gs`: GAS作成アカウント、通知先、シート名などの設定
- `gas/Code.gs`: 投稿受付、一覧取得、スプレッドシート保存、メール通知を行うGASコード
- `gas/appsscript.json`: GASの設定ファイル
- `docs/setup.md`: 初期設定手順

## 初期設定の流れ

1. `kazutoshi_ishikawa@nishikamakura-jichikai.com` でGoogle Apps Scriptプロジェクトを作ります。
2. `gas/Config.gs`, `gas/Code.gs`, `gas/appsscript.json` をGASへ入れます。
3. GASで `setupMobilityNeedsBoard()` を実行し、保存用スプレッドシートを作ります。
4. GASをWebアプリとしてデプロイします。
5. `app.js` の `GAS_WEB_APP_URL` にWebアプリURLを入れます。
6. GitHubリポジトリ `Mobility_Dest_Enq` のGitHub Pagesで公開します。

詳しくは [docs/setup.md](docs/setup.md) を見てください。

## 投稿データの保存先

投稿内容は、GASで `setupMobilityNeedsBoard()` を実行したときに自動作成されるGoogleスプレッドシート「移動しやすいまち西鎌倉 投稿ボード」の `投稿` シートに保存されます。確認・変更用リンクは同じスプレッドシートの `管理リンク` シートに保存されます。既存のスプレッドシートを使う場合は、GASで `connectSpreadsheet('スプレッドシートID')` を実行して接続します。事務局は `投稿` シートの「公開状態」を `公開` にするとWebへ公開でき、`非公開` にすると公開を取り消せます。

## 必須項目と入力上限

次の項目はすべて必須です。`公開状態` が `公開` になった投稿だけがWebに表示されます。`メールアドレス` と `記入者` はWeb公開されません。

- 具体的な場面: 2000文字
- メールアドレス: 254文字
- 目的の内容: 200文字
- 出発地、行き先、聞いた相手: 160文字
- 地域、時間帯、移動する人、記入者: 120文字
- ニックネーム: 80文字

## セキュリティメモ

- 公開一覧の表示時は、投稿内容をHTMLとして挿入せず、`escapeHtml()` でエスケープしています。
- GAS側でも制御文字を除去し、文字数を切り詰めています。
- スプレッドシートで数式として解釈されないよう、`=`, `+`, `-`, `@` で始まる入力には先頭に `'` を付けています。
- 確認・変更用リンクはメールアドレスとトークンで確認し、有効期限は30日です。

## 反映ルール

GitHub PagesやGoogle Apps Scriptへ反映する前に、必ず差分を確認し、内容を要約して承認を得ます。
